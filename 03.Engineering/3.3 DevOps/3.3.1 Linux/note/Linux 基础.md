# 概要
---

Linux 是一个开源的 [[类UNIX操作系统]] 。该系统内核由 Linus Benedict Torvalds 于1991年首次发布。

Linux 严格来说仅表示由 Linus 开发的操作系统的 [[内核]]。而一个操作系统除内核之外，包含了许多软件和实用工具。

如今由于大家在称呼上的习惯，Linux 被用来代指基于 [[Linux 内核]] 的完整的操作系统，而内核则改以 [[Linux 内核]] 称之。
