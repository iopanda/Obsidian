# 概念

在 Power Query 中，可以创建一个表，其中包含列中每个唯一值的聚合值。 Power Query对每个唯一值进行分组，对每个值执行聚合计算，并将列透视到新表中。

# 用法

透视即将表中某一列的列值作为属性到多列中。透视会增加列数。透视后的表符合人类的阅读习惯。
