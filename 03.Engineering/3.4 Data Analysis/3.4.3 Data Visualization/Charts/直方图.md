# 概念

直方图适合用来显示在连续间隔或特定时间段内的数据分布，当中每个条形表示每个间隔/时间段中的频率。直方图的总面积也相等于数据总量。

直方图有助于估计数值集中位置、上下限值以及确定是否存在差距或异常值；也可粗略显示概率分布。

# 功能

比较，数据随时间的变化，分布，模式，范围
