# 概念

雷达图是用来进行多个指标体系比较分析的专业图表，从雷达图中可以看出指标的实际值与参照值的偏离程度，从而为分析者提供有益的信息。

# 作用

雷达图一般用于成绩展示、效果对比量化、多维数据对比等等，只要有前后2组3项以上数据均可制作雷达图，其展示效果非常直观，而且图像清晰耐看，而且制作的雷达图非常漂亮，而且还可个性美化。
