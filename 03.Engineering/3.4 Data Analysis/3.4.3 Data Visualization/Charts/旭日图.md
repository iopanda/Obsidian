# 概念

旭日图Sunburst Chart，是饼图的变形，也可以看作是多级饼图，它可以展示多个层次的比例结构，并能够清晰展示出不同层次数据间的关系。

# 用途

展示不同分类的占比情况时，我们经常用到的是饼图，但遇到多层次的数据时，饼图就显得力不从心，这时“旭日图”就更加适合。
