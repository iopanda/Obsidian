# 概念

Amazon Cognito 可以为Web和移动应用程序提供身份验证、授权和用户管理。用户可以使用用户名和密码直接登录，也可以通过第三方（如Facebook、Amazon、Google或Apple等）登录。

Amazon Cognito 的两个主要组件是用户池和身份池。
- [[Amazon Cognito 用户池]] 是为应用程序提供注册和登录选项的用户目录。
- [[身份池]] 可以用于授予用户访问其他AWS服务的权限。
