# Overview
---
[[AWS Organizations]] 是用于分组和集中管理企业拥有的 AWS 账户 的服务。如果在组织内启用了所有功能，则可对任意或全部账户应用服务控制策略 (SCP)。SCP 是指定[[组织]]或[[组织单元 (OU)]] 的最大权限的 JSON 策略。SCP 限制成员账户中实体（包括每个 [[AWS 账户根用户]]）的权限。任一项策略中的显式拒绝将覆盖允许。
