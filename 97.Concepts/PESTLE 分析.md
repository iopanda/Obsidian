# 概念

PESTLE 分析是一种结构化的宏观分析工具，包括：
- 政治：Political
- 经济：Economic
- 社会：Social
- 技术：Technological
- 法律：Legal
- 环境：Environmental




# 用法

作为战略框架，PESTLE分析可以用于更好地了解哪些直接影响组织未来趋势的因素，如：人口统计、政治障碍、颠覆性技术和竞争压力。在开办新业务或进入新的国外市场时，该工具特别有用。
