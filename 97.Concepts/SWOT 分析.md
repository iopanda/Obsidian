# 概念

SWOT 由四个英文单词首字母组成，分别是：
- 优势：Strengths
- 劣势：Weaknesses
- 机会：Opportunities
- 威胁：Threats


# 用法

可以利用机会形成优势，或者确保自己免受威胁，但是很难改变机会和威胁，如很难改变竞争对手、原材料价格和客户的生活方式。可以采用有效和适当的管理流程来降低这些风险。
